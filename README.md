# React Pizza

Приложение для заказа пиццы на React

## Команды

Доступные команды:

### `npm start`

Запуск приложения в dev режиме

### `npm test`

Запуск тестов

### `npm run build`

Сборка проекта в production

### `npm run lint`

Проверка линтерам и prettier-ом

### `npm run lint:fix`

Исправление ошибок с помощью линтеров и prettier

### `npm run typehint`

Проверка типов Typescript
